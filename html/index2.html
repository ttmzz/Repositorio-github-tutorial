Como clonar um repositório do GitHub

1. Copie o link do repositório

  • Acesse o repositório no GitHub, clique no botão verde "Código", e copie a URL exibida.

2. Abra o terminal ou prompt de comando

  • Vá para o diretório desejado no seu computador.

2.1Use o comando : git clone

  • Digite o comando abaixo no terminal: git clone <URL>

3. Aguarde o processo terminar

  • O Git irá baixar os arquivos para o seu ambiente local.

4. Acesse a pasta do repositório

  • Após a clonagem, utilize o comando: cd nome-do-repositorio
