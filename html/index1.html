Como criar um repositório e enviá-lo ao GitHub

1. Criar um repositório no GitHub

  • Acesse o GitHub e faça login.
  • Clique em "New repository" ou no botão "+".
  • Escolha um nome para o repositório e configure a visibilidade.
  • Clique em "Create repository".

2. Preparar seu projeto localmente

  • No terminal, navegue até a pasta onde está seu projeto.
  • Inicie o Git no diretório com: git init.

3. Adicionar e confirmar os arquivos

  • Adicione os arquivos ao Git com: git add ..
  • Confirme as alterações com: git commit -m "Primeiro commit".

4. Conectar ao repositório remoto

  • Copie a URL do repositório do GitHub.
  • Conecte o repositório remoto com: git remote add origin URL_DO_REPOSITORIO.

5. Enviar os arquivos para o GitHub

  • Envie os arquivos para o repositório remoto com: git push -u origin main.

Alterar Entre Repositórios do GitHub

Passo 1: Verifique o remoto atual

  • git remote -v

Passo 2: Atualize a URL

  • git remote set-url origin <novo_url>
