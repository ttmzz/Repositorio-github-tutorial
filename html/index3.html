Criar, verificar e apontar para uma nova branch

1. Para criar uma nova branch

  • Use o comando: git branch nome-da-branch
  • Isso criará uma nova branch, mas você ainda estará na branch atual.

2. Verificar a lista de branches

  • Para ver todas as branches no seu repositório, use: git branch
  • A branch ativa será marcada com um asterisco (*).

3. Mudar para a nova branch

  • Para trocar para a nova branch e começar a trabalhar nela: git checkout nome-da-branch
  • Ou, se estiver usando Git versão 2.23 ou superior, pode usar: git switch nome-da-branch

4. Criar e já mudar para a nova branch (atalho)

  • Em um único comando, crie e mude para a nova branch: git checkout -b nome-da-branch
  • Ou, para versões mais recentes do Git: git switch -c nome-da-branch

5. Se precisar enviá-la para um repositório remoto

  • Use: git push --set-upstream origin nome-da-branch
